# 豆瓣+图书馆查询助手

一款强大的浏览器扩展，让您在浏览豆瓣图书时自动查询全国各大图书馆的馆藏信息。快速了解图书的借阅状态、索书号和所在位置，轻松找到所需的图书资源，大幅提升图书查询效率。

## 功能特点

### 核心功能
- **自动查询**：浏览豆瓣图书页面时，自动显示所选图书馆的馆藏信息
- **多馆支持**：支持全国60+家省市级图书馆，持续增加中
- **实时状态**：显示图书的借阅状态（在馆/借出）和可借数量
- **详细信息**：提供索书号、所在位置、出版社等完整图书信息

### 新增功能

#### 一键复制查询信息
- 在豆瓣图书页面，每个图书馆信息框右上角均设有"复制"按钮
- 点击即可一键复制该图书馆的完整查询信息
- 复制内容包含：图书馆名称、书名、索书号、馆藏状态等
- 复制成功后按钮显示"已复制"，2秒后自动恢复

#### 右键自动查书
- 在任意网页选中文本（书名或ISBN）
- 右键点击，选择"查书 '选中的文本'"菜单项
- 系统自动弹出查书窗口并查询馆藏信息
- 支持长文本自动换行显示，确保信息清晰可读

## 使用方法

### 基础使用
1. 点击浏览器工具栏中的插件图标
2. 点击界面中的齿轮图标进入设置页面
3. 选择您感兴趣的省市图书馆
4. 关闭设置页面，插件将自动加载所选图书馆信息

### 高级功能
- **一键复制**：在豆瓣图书页面，点击信息框右上角的"复制"按钮即可复制查询信息
- **右键查书**：在任意网页选中文本后右键，选择"查书"菜单项进行快速查询
